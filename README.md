# Preserve Container Items
Gear items that are at 0% condition will no longer disappear from containers.

⚠️ CAUTION:  Even though I have tested this with many different gear items and different containers, it is possible some type of container may have some unusual characteristics that might prevent this mod from working correctly.  So, before you go filling a container with ruined food, it might be best to test with a single item first and make sure it does not dissapear.  You have been warned.   ⚠️
